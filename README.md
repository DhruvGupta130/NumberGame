# 🎲 Number Guessing Game

Welcome to the **Number Guessing Game**! This fun and interactive game challenges you to guess a randomly selected number between 1 and 100 within a limited number of attempts. Can you guess it correctly? Let’s find out! 🎉

## 🌟 Features

- 🎮 Simple and engaging gameplay
- 🔢 Random number generation between 1 and 100
- ⏳ Limited attempts (5 trials)
- 💡 Hints provided for each guess
- 🚀 Easy to play from the command line

## 🛠️ Prerequisites

Before you start, ensure you have:

- Java Development Kit (JDK) installed on your machine
- Basic understanding of Java (optional, for modifications)

## 📥 Installation

To get started with the Number Guessing Game, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/number-guessing-game.git
   ```

2. Navigate to the project directory:
   ```bash
   cd number-guessing-game
   ```

3. Compile the Java program:
   ```bash
   javac game.java
   ```

4. Run the program:
   ```bash
   java game
   ```

## 🎮 How to Play

1. Start the game by choosing to continue after reading the rules.
2. Guess the number between 1 and 100.
3. You have 5 attempts to guess the correct number.
4. After each guess, you will receive hints to help you.
5. Try to guess the number correctly before running out of attempts! 🏆

## 🧩 Game Logic

- The game randomly selects a number between 1 and 100.
- Players have 5 trials to guess the number.
- After each guess, the player is informed if the actual number is higher or lower than their guess.
- If the player guesses correctly, a congratulatory message is displayed along with the number of attempts taken.

## 🎨 Customization

You can customize the game by:

- **Changing the number range**: Modify the random number generation line to adjust the limits.
- **Updating the number of trials**: Change the trial limit to make the game easier or harder.
- **Adding more features**: Implement scoring or advanced hints!
